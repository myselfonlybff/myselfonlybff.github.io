# myselfonlybff.github.io

Ce projet est configuré pour GitHub Pages avec la logique du jeu majoritairement en Python via PyScript.

## Fichiers à pousser

- `index.html`
- `README.md`

## Déploiement GitHub Pages

1. Vérifie que `index.html` est bien présent à la racine du dépôt.
2. Commit et push vers la branche `main` ou `master`.
3. Active GitHub Pages dans les paramètres du dépôt en utilisant la branche choisie et le dossier racine (`/`).
4. Le site sera accessible sur : `https://myselfonlybff.github.io/`

## Notes

- `app.py` n’est pas utilisé par GitHub Pages.
- Tout le comportement du jeu est exécuté dans le navigateur via Python et PyScript.
- Le score est sauvegardé localement dans le navigateur avec `localStorage`.


Ce projet est maintenant prêt pour GitHub Pages.

## Déploiement GitHub Pages

1. Assure-toi que le fichier racine s'appelle `index.html`.
2. Supprime ou ignore `app.py` si tu n’utilises pas de serveur Python pour GitHub Pages.
3. Fais un commit et un push vers la branche `main` ou `master` du dépôt GitHub.
4. Active GitHub Pages dans les paramètres du dépôt en choisissant la branche et le dossier racine (`/`).
5. Ouvre ton site à :
   `https://myselfonlybff.github.io/`

## Notes

- Le jeu fonctionne en JavaScript côté client.
- GitHub Pages ne peut pas exécuter `app.py`.
- Le score est sauvegardé localement dans le navigateur via `localStorage`.
